# 🚐 Travel Trucks

Web application for browsing and filtering campers with the ability to view detailed information and create bookings. Data is fetched from a REST API.

## 📌 Features

- Campers list with pagination (`Load more`)
- Filtering:
  - Location
  - Body type
  - Engine type
  - Transmission
- Camper details page with image slider
- Booking form with validation
- Toast notifications
- State handling:
  - Loading
  - Error
  - Empty results

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Axios
- React Query
- Formik + Yup
- Swiper
- React Hot Toast
- React Icons
- CSS Modules

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/travel-trucks.git
cd travel-trucks
npm install
npm run dev
```

## ▶️ Usage

- Open the catalog and apply filters
- Select a camper to view details and images
- Submit a booking request via the form
